# Step 步骤条

步骤条组件，用于引导用户按照流程完成任务的导航条。

## 基础用法

展示基础的步骤条用法。

<Steps type="basic" size="small" current={1}>
  <Steps.Step title="Step 1" />
  <Steps.Step title="Step 2" description="This is a description" />
  <Steps.Step title="Step 3" />
</Steps>

```tsx
import { Steps } from '@@coze/coze-design';

<Steps type="basic" size="small">
  <Steps.Step title="Step 1" />
  <Steps.Step title="Step 2" description="This is a description" />
  <Steps.Step title="Step 3" />
</Steps>;
```

- 支持基础步骤展示
- 继承自 Semi Design 的 Steps 组件，保持了其核心功能
- 优化了 description 为空时的StepItem的垂直居中样式
