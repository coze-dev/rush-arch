## 组件引入
可从 `@coze-arch/coze-design` 中引入 `SegmentTab` 组件，参考如下：

```tsx
import { SegmentTab } from '@coze-arch/coze-design';
```

## 交互预览
`SegmentTab` 组件支持如下基础属性配置，包含有: `size`, `value` 和 `defaultValue`，可通过调整Control参数来预览。

## 尺寸
> `SegmentTab` 组件支持两种预设尺寸，包含有: `default` 和 `small`。
<Story of={SegmentTabStories.Default} />

- **Container** ：`display: flex` | `gap: 4px` | `padding: 4px` | `border-radius: 12px`

- **Tab** ：`display: inline-flex` | `height: 32px` | `gap: 6px` | `padding: 6px 15px` | `border-radius: 8px`

```tsx
import { SegmentTab } from '@coze-arch/coze-design';

<SegmentTab defaultValue={'Tab1'}  options={['Tab1', 'Tab2', 'Tab3']} />
```

<Story of={SegmentTabStories.Size} />

- **Container** ：`display: flex` | `gap: 4px` | `padding: 4px` | `border-radius: 10px`

- **Tab** ：`display: inline-flex` | `height: 24px` | `gap: 4px` | `padding: 4px 10px` | `border-radius: 8px`

```tsx
import { SegmentTab } from '@coze-arch/coze-design';

<SegmentTab defaultValue={'Tab1'} size="small"  options={['Tab1', 'Tab2', 'Tab3']} />
```

## Props
`SegmentTab`通过类型工具Pick提取了部分`radioGroup`属性使用，详见如下内容：

| Prop         | Type                                | Description           | Default   |
|:-------------|:------------------------------------|:----------------------|:----------|
| size         | `ButtonSize`                        | Sets the size         | `default` |
| className    | `string`                            | Sets the className    |           |
| options      | `string[]` or `Array<OptionItem>`   | Sets the options      | `[]`      |
| value        | `string` or `number`                | Sets the value        |           |
| defaultValue | `string` or `number`                | Sets the defaultValue |           |
| onChange     | `(event: RadioChangeEvent) => void` | Sets the value change |           |
| style        | `React.CSSProperties`               | Sets the custom style |           |


## Types
SegmentTab扩展类型参考如下：

```tsx
export type Size = 'default' | 'small'
```

```tsx
export interface SegmentTabProps
  extends Pick<
      RadioGroupProps,
      'options' | 'defaultValue' | 'className' | 'onChange' | 'value' | 'style'
    > {
  size?: 'default' | 'small';
}
```
