## 组件引入

可从 `@coze-arch/coze-design` 中引入 `Button` 组件，参考如下：

```tsx
import { Button } from '@coze-arch/coze-design';
```

## 交互预览

`Button` 组件支持如下基础属性配置，包含有: `color`, `size`, `color`, `disabled` , `loading` 和 `iconPosition`，可通过调整Control参数来预览。

## 组件类型

`Button` 组件支持五种基础类型，包含有: `Button`, `IconButton`, `LoadingButton`, `AIButton` 和 `SplitButton`，可根据实际情况灵活选用。

- **Button** ：基础button组件，支持传入文字和图标组合使用；
- **IconButton** ：button中仅有icon组件的形式，适合作为图标按钮使用，组件样式做等高等宽处理；
- **LoadingButton** ：button的加载提示样式，加载中会有toast文字提示；
- **AIButton** ：针对AI场下的Button定制样式，按钮字体和背景均做整体做渐变色处理，固定图标样式；
- **SplitButton**： 分组按钮，包含边的文字和右边的下拉按钮部分组成，可响应不同的点击事件；

<Story of={ButtonStories.Buttons} />

```tsx
import {
  Button,
  IconButton,
  LoadingButton,
  AIButton,
  SplitButton,
} from '@coze-arch/coze-design';

<View>
  <IconButton />
  <Button>COZButton</Button>
  <LoadingButton>LoadingButton</LoadingButton>
  <AIButton>LoadingButton</AIButton>
  <SplitButton>SplitButton</SplitButton>
</View>;
```

## 色彩

> `Button` 组件支持七种预设色值，包含有: `hgltplus`, `highlight`, `primary`, `secondary`, `red`,`redhglt`, `green` 和 `yellow`。

<Story of={ButtonStories.Colors} />

```tsx

<View>
  <Button color="hgltplus">hgltplus</Button>
  <Button color="highlight">highlight</Button>
  <Button color="primary">primary</Button>
  <Button color="secondary">secondary</Button>
  <Button color="yellow">yellow</Button>
  <Button color="green">green</Button>
  <Button color="red">red</Button>
  <Button color="redhglt">redhglt</Button>
</View>;
```

> 针对AI功能场景，预设有三种定制色彩：`aiplus`, `aihglt` 和 `aiprimary`。

<Story of={ButtonStories.WithAI} />

```tsx
import { AIButton } from '@coze-arch/coze-design';

<View>
  <AIButton color="hgltplus">hgltplus</AIButton>

```

如仅需使用按钮，也支持隐藏默认图标：`hideIcon`。

<Story of={ButtonStories.HideAIIcon} />

## 尺寸

> `Button` 组件支持四种预设尺寸，包含有: `large`, `default`, `small` 和 `mini`。

<Story of={ButtonStories.Sizes} />

- **large** ：`height: 40px` | `min-width: 64px` | `padding: 9px 20px` | `border-radius: 10px`
- **default** ：`height: 32px` | `min-width: 56px` | `padding: 6px 15px` | `border-radius: 8px`
- **small** ：`height: 24px` | `min-width: 42px` | `padding: 4px 10px` | `border-radius: 6px`
- **mini** ：`height: 16px` | `min-width: auto` | `padding: 0px 5px` | `border-radius: 4px`

```tsx
import { Button } from '@coze-arch/coze-design';

<View>
  <Button size="large">large</Button>
  <Button size="default">default</Button>
  <Button size="small">small</Button>
  <Button size="mini">mini</Button>
</View>;
```

## 图标

> `Button` 组件支持搭配图标使用，可通过: `icon` 字段传入自定义图标，同时支持 `iconPosition` 来自定义图标在按钮中相对文字的位置。

<Story of={ButtonStories.WithIcon} />

- **icon** ：可通过icon组件导出自定义组件，传入 `icon={<IconCozPeopleFill />}`
- **iconPosition** ：支持左向 `iconPosition="left"` 和 右向`iconPosition="right"`

```tsx
import { Button } from '@coze-arch/coze-design';
import { IconCozPeopleFill } from '@coze-arch/arco-icon';

<View>
  <Button icon={<IconCozPeopleFill />}>with icon</Button>
  <Button icon={<IconCozPeopleFill />} iconPostion="left">
    icon left
  </Button>
  <Button icon={<IconCozPeopleFill />} iconPostion="right">
    icon right
  </Button>
</View>;
```

## 事件

> `Button` 组件支持`onClick`, `onMouseEnter`, `onMouseLeave`, `onFocus`, `onBlur`等标准事件使用。

<Story of={ButtonStories.WithState} />

- **onClick** ：对应HTMLDivElement中的标准事件。

```tsx
import { Button } from '@coze-arch/coze-design';
import { useState } from 'react';

const [isOpen, setIsOpen] = useState(false);
const handlePress = () => {
  setIsOpen(prev => !prev);
};

return (
  <View prop="variant" value="state" justify="start">
    <Button aria-label="Open" aria-pressed={isOpen} onClick={handlePress}>
      click change
    </Button>
    <div className="coz-fg-plus mt-1">state: {isOpen ? 'close' : 'open'}</div>
  </View>
);
```

## Props

所有semi的按钮属性都可以在`COZButton`组件上使用，此外还支持以下属性：

| Prop         | Type              | Description             | Default   |
| :----------- | :---------------- | :---------------------- | :-------- |
| color        | `ButtonColor`     | Sets the color          | `brand`   |
| size         | `ButtonSize`      | Sets the size           | `default` |
| disabled     | `Boolean`         | Sets the disabled state | `false`   |
| loading      | `Boolean`         | Sets the loading state  | `false`   |
| iconPosition | `left` or `right` | Sets the size           | `left`    |
