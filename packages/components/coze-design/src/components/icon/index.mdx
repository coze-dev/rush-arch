## 组件引入
可从 `@coze-arch/coze-design/icons` 中引入对应所需图标，参考如下：

```tsx
import { IconCozPlusFill } from '@coze-arch/coze-design/icons';
```

生成svg节点预览，默认fill为`currentColor`, height为`1em`, width为`1em` 。

```html
<svg class="icon-icon icon-icon-coz_people_fill text-lg" width="1em" height="1em" viewBox="0 0 24 24" fill="currentColor" xmlns="http://www.w3.org/2000/svg"><path d="M15 ....></path></svg>
```

## 尺寸
`Icon` 的尺寸可以通过className的方式传入控制，对应为字号大小，如：`text-lg`, `text-base`, `text-xxl`。

- **text-xxl** ：对应`font-size: 16px`, 应用到图标后为：`width:16x` `height: 16px` ；
- **text-lg** ：对应`font-size: 14px`, 应用到图标后为：`width:14px` `height: 14px` ；
- **text-base** ：对应`font-size: 12px`, 应用到图标后为：`width:12px` `height: 12px` ；

<Story of={IconStories.Size} />

```tsx
import { IconCozPlusFill } from '@coze-arch/coze-design/icons';

<>
  <IconCozPeopleFill className='text-xxl' />
  <IconCozPeopleFill className='text-lg' />
  <IconCozPeopleFill className='text-base' />
</>
```

## 色彩
`Icon` 的色彩可以通过className的方式传入控制，可参考`token`中的预设色值，如：`coz-fg-hglt`, `coz-fg-hglt-green`, `coz-fg-hglt-red`。

<Story of={IconStories.Color} />

```tsx
import { IconCozPlusFill } from '@coze-arch/coze-design/icons';

<>
   <IconCozPeopleFill className='coz-fg-hglt' />
   <IconCozPeopleFill className='coz-fg-hglt-red' />
   <IconCozPeopleFill className='coz-fg-hglt-green' />
</>
```

## 图标按钮
`Icon` 通常可以搭配`IconButton`组件来组合使用，适用于给一个SVG图标加入一些`onClick`事件或者`hover`、`active` 的交互样式效果，这里建议同时传入色彩和尺寸样式，精准控制大小和色彩显示。

<Story of={IconStories.IconBtn} />

```tsx
import { IconCozPlusFill } from '@coze-arch/coze-design/icons';

<>
  <IconButton icon={<IconCozPeopleFill className='text-xxl coz-fg-hglt' />} />
</>
```

## Props
| Prop      | Type     | Description           | Default |
|:----------|:---------|:----------------------|:--------|
| height    | `Number` | Sets the width        | `1em`   |
| width     | `Number` | Sets the height       | `1em`   |
| className | `String` | Sets the avatar class |         |
