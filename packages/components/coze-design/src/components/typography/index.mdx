## 组件引入
可从 `@coze/coze-design` 中引入 `Badge` 组件，参考如下：

```tsx
import { Typography } from '@coze/coze-design';
```

## 交互预览

## 类型
`Typography` 支持多种类型，包括：`Title`, `Text`, `Paragraph` 和 `Numeral`。
- **Title** ：标题组件, 可通过设置 heading 可以展示不同级别的标题，coze-design 扩展了fontSize字段，支持部分预设尺寸;
- **Text** ：文本组件，支持semi的`Typography.Text`的所有属性；
- **Paragraph** ：段落组件，支持semi的`Typography.Paragraph`的所有属性, 可以通过设置 spacing='extended' 使用更宽松的行距；
- **Numeral** ：数值组件，支持semi的`Typography.Numeral`的所有属性, 在Text组件的基础上提供需要单独处理文本中数值的能力；

<Story of={TypographyStories.Type} />

```tsx
import { Typography } from '@coze/coze-design';

const { Title, Text, Paragraph, Numeral } = Typography;

<Title>标题Title</Title>

<Text>文字Text</Text>

<Paragraph>段落Paragraph</Paragraph>

<Numeral precision={2}>
  数字Numeral：1.6111e1 K
</Numeral>
```

## 标题
`Title` 的 `fontSize`支持 `28px`、`20px`、`16px`、`14px`、`12px` 和 `inherit`，当与 `heading` 属性同时使用时，`fontSize`会先生效，默认为`inherit`，兼容semi的`Typography.Title`的所有属性。
<Story of={TypographyStories.Title} />

`Title`的 **fontSize** 和 **leading** 对应默认对应值参考如下，如需定制可以通过：`className` 传值覆盖，如：`className='!text-22px !leading-24px'`
- **28px** ：`font-size: 28px` | `line-height: 36px`
- **20px** ：`font-size: 20px` | `line-height: 28px`
- **16px** ：`font-size: 16px` | `line-height: 22px`
- **14px** ：`font-size: 14px` | `line-height: 20px`
- **12px** ：`font-size: 12px` | `line-height: 16px`

```tsx
import { Typography } from '@coze/coze-design';

const { Title: COZTitle } = Typography;

<div>
  <COZTitle heading={1}>默认标题（H1）</COZTitle>
  <COZTitle heading={2}>默认标题（H2）</COZTitle>
  <COZTitle heading={3}>默认标题（H3）</COZTitle>
  <COZTitle heading={4}>默认标题（H4）</COZTitle>
</div>

<div>
  <COZTitle fontSize="28px">COZTitle28（特大标题）</COZTitle>
  <COZTitle fontSize="20px">COZTitle20（大标题）</COZTitle>
  <COZTitle fontSize="16px">COZTitle16（标题）</COZTitle>
  <COZTitle fontSize="14px">COZTitle14（小标题）</COZTitle>
  <COZTitle fontSize="12px">COZTitle12（特小标题）</COZTitle>
</div>

<div className="flex flex-col gap-2">
  <COZTitle className='coz-fg-primary !text-22px'>COZTitle22（自定义22px字号）</COZTitle>
  <COZTitle fontSize="20px" className='coz-fg-hglt-green'>COZTitle20（自定义颜色）</COZTitle>
  <COZTitle fontSize="16px" className='coz-fg-primary !leading-28px'>COZTitle16（自定义行高）</COZTitle>
</div>

```

## 文本
`Text` 的`fontSize`支持 `16px`、`14px`、`12px`、`10px` 和 `inherit`，默认为`inherit`，兼容semi的`Typography.Text`的所有属性。
<Story of={TypographyStories.Text} />

`Text`的 **fontSize** 和 **leading** 对应默认对应值参考如下，如需定制可以通过：`className` 传值覆盖，如：`className='!text-xl !leading-20px'`
- **16px** ：`font-size: 16px` | `line-height: 28px`
- **14px** ：`font-size: 14px` | `line-height: 20px`
- **12px** ：`font-size: 12px` | `line-height: 16px`
- **10px** ：`font-size: 10px` | `line-height: 14px`


```tsx
import { Typography } from '@coze/coze-design';

const { Text: COZText } = Typography;

<div>
   <COZText size="normal">默认文字normal</COZText>
   <COZText size="small">默认文字small</COZText>
   <COZText size="inherit">默认文字inherit</COZText>
</div>

<div>
   <COZText fontSize="16px">COZText16（大文字）</COZText>
   <COZText fontSize="14px">COZText14（标准文字）</COZText>
   <COZText fontSize="12px">COZText12（小文字）</COZText>
   <COZText fontSize="10px">COZText10（特小文字）</COZText>
</div>

```

## 段落
`Paragraph` 的`fontSize`支持 `16px`、`14px`、`12px` 和 `inherit`，默认为`inherit`，兼容semi的`Typography.Paragraph`的所有属性。
<Story of={TypographyStories.Paragraph} />

`Paragraph`的 **fontSize** 和 **leading** 对应默认对应值参考如下，如需定制可以通过：`className` 传值覆盖，如：`className='!text-xl !leading-20px'`
- **16px** ：`font-size: 16px` | `line-height: 28px`
- **14px** ：`font-size: 14px` | `line-height: 20px`
- **12px** ：`font-size: 12px` | `line-height: 16px`

```tsx

const { Paragraph: COZParagraph } = Typography;

<COZParagraph
    ellipsis={{
      rows: 3,
      expandable: true,
      collapsible: true,
      collapseText: '折叠我吧',
      onExpand: (bool, e) => console.log(bool, e),
    }}
    style={{width: 300}}
>
  支持展开和折叠：Coze Design
  设计系统包含设计语言以及一整套可复用的前端组件，帮助设计师与开发者更容易地打造高质量的、用户体验一致的、符合设计规范的
  Web 应用。
</COZParagraph>

```

## 数值
`Numeral` Numeral 组件在Text组件的基础上，添加了属性: `rule`, `precision`, `truncate`, `parser`, 以提供需要单独处理文本中数值的能力。
<Story of={TypographyStories.Numeral} />

```tsx
import { Typography } from '@coze/coze-design';

const { Numeral: COZNumeral } = Typography;

<COZNumeral className="coz-fg-primary" precision={2}>
  <p>点赞量：1.6111e1 K</p>
</COZNumeral>

<COZNumeral
  rule="percentages"
  className="coz-fg-primary"
  precision={2}
>
  <p>好评率: 0.915</p>
</COZNumeral>

 <COZNumeral rule="percentages" className="coz-fg-primary">
    这场比赛我的胜率是0.6，输的概率是0.4
 </COZNumeral>
```

## 溢出省略
`ellipsis` 可通过配置设置自动溢出省略。
<Story of={TypographyStories.Ellipsis} />

```tsx
import { Typography } from '@coze/coze-design';

const { Paragraph: COZParagraph } = Typography;

<COZParagraph
    fontSize="14px"
    ellipsis={{
      rows: 3,
      showTooltip: {
        type: 'tooltip', // tooltip | popover
        opts: {
          content: '测试截断文字传入',
        },
      },
    }}
    style={{width: 500}}
  >
    Coze Design 设计系统包含设计语言以及一整套可复用的前端组件，帮助设计师与开发者更容易地打造高质量的、用户体验一致的、符合设计规范的
    Web 应用。
</COZParagraph>
```

## Props
所有semi的`typography`属性都可以在`COZTypography`组件上使用，此外还支持以下属性：

| Prop      | Type       | Description          | Default   |
|:----------|:-----------|:---------------------|:----------|
| fontSize  | `BaseSize` | Sets the fontSize    | `inherit` |
| className | `String`   | Sets the badge class |           |

## Types
COZBadge 扩展类型参考如下：

```tsx
export type TitleSize = '28px' | '20px' | '16px' | '14px' | '12px' | 'inherit';
```

```tsx
export type TextSize = '16px' | '14px' | '12px' | '10px' |  'inherit' ;
```
