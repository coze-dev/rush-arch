## 组件引入
可从 `@coze/coze-design` 中引入 `Progress` 组件，参考如下：

```tsx
import { Progress } from '@coze/coze-design';
```

## 交互预览
`Progress` 组件支持如下基础属性配置，包含有: `size`, `type`, `height`，可通过调整Control参数来预览。

## 类型
> `Progress` 组件支持两种预设类型，包含有: `line` 和 `circle`，默认为`line`。
<Story of={ProgressStories.Line} />

- **style** ：可以通过 `style` 传入 `height` 属性自定义进度条高度。
- **stroke** ：可以通过 `stroke` 传入 `coze token` 属性自定义进度条颜色。

```tsx
import { Progress } from '@coze/coze-design';

<Progress type='line' size='default' />
<Progress type='line' size='default' style={{ height: 4 }} />
<Progress type='line' size='default' stroke='var(--coz-fg-hglt-green)' />
```

<Story of={ProgressStories.Circle} />

- **width** ：可以通过 `width` 传入自定义宽度数值来控制进度条的外圆形直径。
- **strokeWidth** ：可以通过 `strokeWidth` 传入自定义宽度数值来控制进度条的内圆环宽度。

```tsx
import { Progress } from '@coze/coze-design';

<Progress type='width' size='default' />
<Progress type='circle' size='default' stroke='var(--coz-fg-hglt-green)' />
```

## 尺寸
> `Progress` 组件支持三种预设尺寸，包含有: `small`, `default` 和 `large`。
<Story of={ProgressStories.Size} />

- **small** ：`height: 3px`
- **default** ：`height: 4px`
- **large** ：`height: 6px`
- **custom** ：`height: 2px`, 可自定义`height`数值

```tsx
import { SegmentTab } from '@coze/coze-design';

<Progress type='line' size='small' />
<Progress type='line' size='default' />
<Progress type='line' size='large' />
<Progress type='line' size='default' style={{ height: 2 }} />

// 当只改 height 时，height={2} 和 style={height: 2} 作用等价
<Progress type='line' size='default' height={2} />
```

## Props
`COZProgress` 整体上兼容semi的`progress`属性使用，详见如下内容：

| Prop        | Type                 | Description                                     | Default       |
|:------------|:---------------------|:------------------------------------------------|:--------------|
| size        | `ProgressSize`       | Sets the size                                   | `default`     |
| type        | `ProgressType`       | Sets the progress type                          | `line`        |
| stroke      | `string`             | Sets the progress stroke color                  | `coz-fg-hglt` |
| strokeWidth | `number`             | Sets the progress stroke width, only for circle | `4px`         |
| width       | `number`             | Sets the progress width, only for circle        | `28px`        |
| height      | `number` or `string` | Sets the progress height, only for line         | `4px`         |
