## 组件引入

可从 `@coze/coze-design` 中引入 `CozInputNumber` 组件，参考如下：

```tsx
import { CozInputNumber, InputNumber } from '@coze/coze-design';
```

**注意，当前存在两个版本的 InputNUmber，新接入业务应使用 CozInputNumber，存量业务按需切换至 CozInputNumber，旧版本 InputNumber 后续不再维护，**

> 相关背景：早期设计的 InputNumber 组件需要自定义实现加减按钮，定制的方法、维护的状态较多，容易出现问题。设计调整后仅需基于semi组件进行样式调整，无需维护其他状态。因此推荐使用新版本组件。考虑到组件应用的业务场景较多，两个版本暂时共存，由业务按需自行切换至新版组件

## 尺寸

可以通过 size 属性设置组件尺寸，支持 default 和 small 两种预设尺寸

<Story of={InputNumberStories.Size} />

```tsx
import { CozInputNumber } from '@coze/coze-design';

<CozInputNumber size='default' />
<CozInputNumber size='small' />

```

## 错误态

可以通过 error 属性设置组件为错误态

<CozInputNumber error={true} />

```tsx
import { CozInputNumber } from '@coze/coze-design';

<CozInputNumber error={true} />;
```

## disabled态

可以通过 disabled 属性设置组件为禁用态

<CozInputNumber disabled={true} value={10} />

```tsx
import { CozInputNumber } from '@coze/coze-design';

<CozInputNumber disabled={true} value={10} />;
```

## Props

所有semi InputNumber的属性都可以在COZInputNumber组件上使用，此外还支持以下属性
| Prop | Type | Description | Default |
|:----------|:--------------|:------------------------|:----------|
| error | `boolean` | Sets the error status | `false` |
