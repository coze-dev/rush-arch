# 前景色 foreground

前景色是用户界面中最常见的颜色类型，主要用于文本、图标和其他前景元素。合理使用前景色可以提高内容的可读性和界面的层次感。

## 使用原则

- **对比度**: 前景色应与背景色形成足够的对比度，确保文本内容清晰可读
- **层次感**: 不同重要程度的内容应使用不同深浅的前景色，形成视觉层次
- **一致性**: 在整个产品中保持前景色使用的一致性，提高用户体验

## 应用场景

- 文本内容（标题、正文、辅助文本等）
- 图标和交互元素
- 强调和突出显示的内容
- 状态指示器（成功、警告、错误等）
