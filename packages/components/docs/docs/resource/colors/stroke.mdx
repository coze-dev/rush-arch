# 描边色 stroke

描边色用于界面中的边框、分割线和轮廓线等元素，是界面结构的重要视觉指引。合理使用描边色可以增强界面的层次感和组织结构，提高用户对界面布局的理解。

## 使用原则

- **轻量化**: 描边色通常应保持轻量，不应过于抢眼或喧宾夺主
- **层次区分**: 不同层级的元素可使用不同深浅的描边色
- **状态表达**: 可通过描边色的变化表达元素的不同状态（如聚焦、激活等）

## 应用场景

- 输入框边框
- 卡片和面板边框
- 表格边框和分割线
- 按钮轮廓
- 分隔符和分割线
- 强调和聚焦状态

## 颜色分类

### 主题色 (Theme)

主题色是与产品主题相关的描边色调，适用于不同场景和状态。包括基础描边色和高亮描边色。

## 描边色层级

描边色通常按照以下层级组织：

- **强层级 (stroke-plus)**: 用于输入框等需要明显边框的组件
- **默认层级 (stroke-primary)**: 用于分界线、分割线等常规边框
- **高亮层级 (stroke-hglt)**: 用于需要强调的边框，如激活状态、聚焦状态等

## 设计技巧

- 在浅色背景上，描边色通常比背景色深一些
- 在深色背景上，描边色通常比背景色浅一些
- 对于需要强调的元素，可使用主题色作为描边色
- 描边粗细与描边色深浅应协调搭配，深色可以更细，浅色可以适当加粗
