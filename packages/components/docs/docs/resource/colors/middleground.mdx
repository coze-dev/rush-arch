# 中景色 middleground

中景色是介于前景色和背景色之间的颜色层次，主要用于界面中的中间层元素，如卡片、面板、分割线等。合理使用中景色可以增强界面的层次感和空间感。

## 使用原则

- **层次区分**: 中景色应与前景色和背景色形成明显的层次区分
- **视觉引导**: 通过中景色的变化引导用户视线和注意力
- **适度对比**: 与背景色保持适度的对比，不应过强或过弱

## 应用场景

- 卡片和面板背景
- 次级导航区域
- 分割线和边框
- 表格行交替色
- 悬浮和选中状态的背景
