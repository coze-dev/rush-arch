# Tag 标签

## 基本信息

### 何时使用

- 需要标记和分类内容时
- 需要标记事物的属性和维度时
- 需要进行简单的状态标记时

### 如何引入

```tsx
import { Tag } from '@coze-arch/coze-design';
```

### 注意事项

- 组件基于 Semi Design 的 Tag 组件封装，继承了其所有功能
- 默认使用了 Coze Design 的主题变量
- 支持两种尺寸：`small`（默认）和 `mini`
- 支持多种预设颜色：`brand`、`primary`、`green`、`yellow`、`red`、`cyan`、`blue`、`purple`、`magenta`
- 支持内置图标和自定义图标
- 支持加载和禁用状态
- 支持点击和关闭事件

## 功能演示

### 基础用法

最基本的标签用法，展示不同尺寸。

```tsx preview
import { Tag } from '@coze-arch/coze-design';
import { IconCozFace } from '@coze-arch/coze-design/icons';

const Demo = () => (
  <div className="flex gap-4">
    <Tag size="small">标准尺寸</Tag>
    <Tag size="mini">迷你尺寸</Tag>
  </div>
);

export default Demo;
```

### 不同颜色

支持多种预设颜色。

```tsx preview
import { Tag } from '@coze-arch/coze-design';
import { IconCozFace } from '@coze-arch/coze-design/icons';

const colors = [
  'brand',
  'primary',
  'green',
  'yellow',
  'red',
  'cyan',
  'blue',
  'purple',
  'magenta',
] as const;

const Demo = () => (
  <div className="flex flex-col gap-4">
    <div>
      <h4>普通状态</h4>
      <div className="flex gap-2 mt-2">
        {colors.map(color => (
          <Tag key={color} color={color}>
            {color}
          </Tag>
        ))}
      </div>
    </div>
    <div>
      <h4>可交互</h4>
      <div className="flex gap-2 mt-2">
        {colors.map(color => (
          <Tag
            key={color}
            color={color}
            onClick={() => console.log('clicked:', color)}
          >
            {color}
          </Tag>
        ))}
      </div>
    </div>
    <div>
      <h4>禁用状态</h4>
      <div className="flex gap-2 mt-2">
        {colors.map(color => (
          <Tag key={color} color={color} disabled>
            {color}
          </Tag>
        ))}
      </div>
    </div>
  </div>
);

export default Demo;
```

### 可交互


```tsx preview

  <div className="flex flex-col gap-4">
```

## API 参考

### Tag Props

| 属性       | 说明         | 类型                                                                                                | 默认值    |
| ---------- | ------------ | --------------------------------------------------------------------------------------------------- | --------- |
| size       | 标签尺寸     | `'small' \| 'mini'`                                                                                 | `'small'` |
| color      | 标签颜色     | `'brand' \| 'primary' \| 'green' \| 'yellow' \| 'red' \| 'cyan' \| 'blue' \| 'purple' \| 'magenta'` | `'brand'` |
| prefixIcon | 前缀图标     | `'info' \| 'clock' \| 'check' \| ReactNode`                                                         | -         |
| suffixIcon | 后缀图标     | `'cross' \| 'info' \| ReactNode`                                                                    | -         |
| loading    | 是否加载中   | `boolean`                                                                                           | `false`   |
| disabled   | 是否禁用     | `boolean`                                                                                           | `false`   |
| closable   | 是否可关闭   | `boolean`                                                                                           | `false`   |
| onClick    | 点击事件回调 | `(e: MouseEvent) => void`                                                                           | -         |
| onClose    | 关闭事件回调 | `(e: MouseEvent) => void`                                                                           | -         |
| className  | 自定义类名   | `string`                                                                                            | -         |
| style      | 自定义样式   | `CSSProperties`                                                                                     | -         |

除了以上属性，Tag 组件还支持原生 Semi Tag 组件的所有属性。 具体请参考 [Semi Tag 文档](https://semi.design/zh-CN/show/tag#API%20参考)。
